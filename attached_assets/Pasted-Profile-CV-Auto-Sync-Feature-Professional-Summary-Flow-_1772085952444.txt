Profile → CV Auto Sync Feature + Professional Summary Flow Fix
🎯 Objective

Now that we store all user data properly in the database (from previous CV steps), we want to improve UX for logged-in users.

When a logged-in user creates a new CV:

They should NOT need to manually re-enter all data again if profile data already exists.

We need to implement a “Sync From Profile” system for each step (except Skills & Certificates).

Also, fix the Professional Summary generation flow.

1️⃣ Add “Sync From Profile” Button (Per Step)
📌 Scope

Add a button on every step EXCEPT:

❌ Skills

❌ Certificates

Add button on:

Basic Information

Location

Professional Targeting

Work Experience

Education

Professional Summary

🟢 Button Label:

“Sync From Profile”

🔁 Behavior

When user clicks “Sync From Profile”:

System fetches data from Profile Settings (latest stored data)

Automatically fills the current step form fields

Does NOT overwrite manually typed data unless confirmed

Show confirmation modal if step already has data:

“Replace current data with profile data?”

🧠 Data Rules

Pull from latest saved profile data

Pull repeater fields properly (Education & Work Experience arrays)

If profile has multiple entries → populate all entries

If some fields are empty → leave them empty (do not inject null)

User must be able to edit synced data

🔐 Important

Only visible for logged-in users

Guest users should NOT see Sync button

Must not create duplicate entries

Must not overwrite entire CV record silently
