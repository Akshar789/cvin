Allow the user to paste their LinkedIn profile URL.
The system will automatically extract publicly available profile data, evaluate it, rewrite it, and give the user a complete improvement report following ATS + LinkedIn SEO standards.

→ Show a popup with:

fetchLinkedInPublicData(linkedinURL)

It should attempt to retrieve only publicly visible fields:

Extract:

Name

Headline

About Summary

Experience (job titles, companies, dates, descriptions)

Education

Licenses & Certifications

Target job domain

Leadership tone
