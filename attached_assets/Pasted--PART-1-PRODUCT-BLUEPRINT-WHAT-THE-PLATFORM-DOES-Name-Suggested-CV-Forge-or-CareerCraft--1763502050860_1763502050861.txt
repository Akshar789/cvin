🟦 PART 3 — REPLIT DEVELOPER FULL SPECIFICATION

(You can copy/paste this to Replit AI Agent or your developer.)

⭐ 1. Tech Stack

Frontend: React / Next.js (Replit-ready templates)

Backend: Node.js (Express server)

Database: Supabase OR Replit DB OR Firebase

Authentication: Supabase Auth or Magic Link

Payments: Stripe

AI Engine: OpenAI API (gpt-4.1-mini & gpt-4.1)

⭐ 2. Database Structure (copy to developer)
Users {
  id: string,
  email: string,
  isSubscribed: boolean,
  freeCredits: number, // default = 3
  createdAt: timestamp
}

UsageLogs {
  userId: string,
  feature: string, // 'cv', 'ats', 'interview', etc.
  date: timestamp,
  tokensUsed: number
}

⭐ 3. API Architecture
POST /generate-cv

Validate user token → Identify user

IF !isSubscribed:

IF freeCredits > 0 → allow

ELSE → return error {code:"SUBSCRIPTION_REQUIRED"}

Call gpt-4.1-mini for free users

Call gpt-4.1 for paid users

Save usage log

Return response

POST /generate-idp

Exact same logic.

POST /generate-interview-questions

Same logic.

POST /ats-check

Same logic.

⭐ 4. Rate Limiting

Add a middleware:

limit: 5 requests per minute per user


Prevents misuse & protects API costs.

⭐ 5. Subscription Activation Workflow
After successful Stripe payment:

Update:

isSubscribed: true
freeCredits: 999 // unlimited

If payment fails:

Revert to:

isSubscribed: false
freeCredits: remaining/free  // If trial not used
