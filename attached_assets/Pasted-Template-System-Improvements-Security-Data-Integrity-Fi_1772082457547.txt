2️⃣ Smart Template Step – Premium Security Fix
❌ Problem:

In Smart Template step:

Preview loads full template

User can download premium template

Even guest user can inspect and remove blur

This is a security issue.

✅ Required Premium Logic
A. For Premium Templates
Logged-in Free User:

Show template preview

Display visible lock icon overlay

Disable download button

Prevent saving template selection

Do NOT allow switching to premium

Show upgrade modal

Guest User:

Apply blur effect on full template

On hover → partially reveal a small area only (visual teaser)

Disable download button

Disable template selection

B. Restrict Inspection & Right Click

In Smart Template step (preview modal only):

Disable right-click

Disable inspect shortcut (F12, Ctrl+Shift+I)

Disable DevTools key combos

Disable copy

Disable drag

Disable image saving

⚠️ Apply only inside preview container.
Do NOT break entire website.

4️⃣ Critical Bug – Dashboard Template Changing
❌ Problem:

User created CV with Template A
Downloaded
Next day logs in
Dashboard shows Template B

This means:

Template ID is being overwritten

Or last preview template is being saved incorrectly

Or auto-save logic updating template incorrectly

✅ Required Fix:

Only save template_id when:

User explicitly selects template

Do NOT save template_id during preview switching

Do NOT auto-save template_id when browsing

Dashboard must display:

Last explicitly selected template

Always fetch template_id from CV record

Do NOT regenerate or override template_id

5️⃣ Critical Bug – Premium Template Save Loophole
❌ Problem:

In CV Preview page:

When free user selects premium template

Template changes visually

Data auto-saves

Premium template might get saved

This is a loophole.

✅ Required Fix:

If user is FREE:

When selecting premium template:

Do NOT update template_id

Do NOT auto-save

Show upgrade modal

Revert to previous template

Prevent any database update

Premium template should NEVER be saved in CV record for free user.

6️⃣ Auto-Save Logic Validation
